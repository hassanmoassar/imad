# Morocco Voyages - Next.js Edition

A luxurious travel website with stunning animations and modern design patterns.

## Features

- ✨ **Creative Animations** - Framer Motion animations throughout the site
- 🎨 **Modern Design** - Tailwind CSS with custom color palette
- 📱 **Responsive** - Mobile-first responsive design
- ⚡ **Performance** - Next.js 14 with optimizations
- 🎯 **SEO Ready** - Meta tags and structured data
- 🌙 **Accessibility** - WCAG compliant

## Animation Features

- Hero section with parallax and fade-in effects
- Service cards with hover animations and rotating icons
- Experience cards with price reveal animations
- Featured section with floating animations
- Contact form with focus state animations
- Footer with rotating background elements
- Scroll-based animations on all sections

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Language**: TypeScript
- **Font**: Playfair Display (serif) & Inter (sans-serif)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser
